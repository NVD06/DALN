"# DALN" 
Web Elearning 
Laravel
Web có các tính năng sau:
Student có thể:
1 Đăng ký, đăng nhập
2 Đăng ký khóa học, xem chi tiết khóa học
3 Đánh giá khóa học
4 Kiểm tra 
5 Trao đổi với giáo viên khóa học
6 Thanh toán

Mentor có thể:
1 Đăng nhập
2 Tạo các khóa học
3 Quản lý các khóa học: thêm, sửa, và xóa các bài giảng trong khóa học


Công nghệ sử dụng
Laravel
